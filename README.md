# dotfiles

Personal dotfiles, managed with [dot][dot].

## Installation

```shell-session
$ cargo install --git https://github.com/ubnt-intrepid/dot.rs.git
$ git clone https://github.com/manoliot/dotfiles.git $HOME/.dotfiles
$ cd .dotfiles
$ ./install
```
